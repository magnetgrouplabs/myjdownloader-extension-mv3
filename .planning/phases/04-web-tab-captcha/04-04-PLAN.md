**Part A: Protocol callbacks in captchaSolverContentscript.js**

Add three new functions and their invocations to the existing content script. All new code uses `var` declarations and function statements (matching IIFE style). Use `XMLHttpRequest` (not fetch).

1. **Declare `canCloseHandle`** at the top alongside `pollingHandle` and `countdownHandle`:
   ```javascript
   var canCloseHandle = null;
   ```

2. **Declare `lastMouseMoveTime`** at the top:
   ```javascript
   var lastMouseMoveTime = 0;
   ```

3. **Add `startCanClosePolling(callbackUrl)` function** after `startCountdown`:
   - `setInterval` at 1000ms (1 second)
   - Each tick: `var xhr = new XMLHttpRequest()`
   - `xhr.open('GET', callbackUrl + '&do=canClose', true)`
   - `xhr.setRequestHeader('X-Myjd-Appkey', 'webextension-' + chrome.runtime.getManifest().version)`
   - `xhr.timeout = 5000`
   - `xhr.onload = function()`: if `xhr.status === 200 && xhr.responseText === 'true'`, clear ALL three intervals (`clearInterval(canCloseHandle)`, `clearInterval(pollingHandle)`, `clearInterval(countdownHandle)`), then try `window.close()`, and as fallback send `chrome.runtime.sendMessage({ action: 'captcha-can-close' })`
   - Return the interval handle
   - Assign: `canCloseHandle = startCanClosePolling(callbackUrl)`

4. **Add `sendLoadedEvent(callbackUrl)` function**:
   - Find CAPTCHA widget element: `document.querySelector('iframe[src*="recaptcha"], iframe[src*="hcaptcha"], .g-recaptcha, .h-captcha')`
   - If not found, retry with `setTimeout(sendLoadedEvent.bind(null, callbackUrl), 500)` — track retry count via closure variable `var loadedRetries = 0` declared at top (max 10 retries)
   - Once found, get `var rect = el.getBoundingClientRect()`
   - Build params: `'&do=loaded' + '&x=' + (window.screenX || window.screenLeft || 0) + '&y=' + (window.screenY || window.screenTop || 0) + '&w=' + (window.outerWidth || 0) + '&h=' + (window.outerHeight || 0) + '&vw=' + (window.innerWidth || 0) + '&vh=' + (window.innerHeight || 0) + '&eleft=' + Math.round(rect.left) + '&etop=' + Math.round(rect.top) + '&ew=' + Math.round(rect.width) + '&eh=' + Math.round(rect.height) + '&dpi=' + (window.devicePixelRatio || 1)`
   - `var xhr = new XMLHttpRequest()`; open GET to `callbackUrl + params`; set `X-Myjd-Appkey` header; `xhr.timeout = 5000`; `xhr.send()`
   - Call `sendLoadedEvent(callbackUrl)` in main flow (after startCountdown, inside isJdLocalhost gate)

5. **Add `startMouseMoveReporting(callbackUrl)` function**:
   - `document.addEventListener('mousemove', function() { ... })`
   - Inside handler: `var now = Date.now(); if (now - lastMouseMoveTime < 3000) return; lastMouseMoveTime = now;`
   - Then: `var xhr = new XMLHttpRequest()`; open GET to `callbackUrl + '&do=canClose&useractive=true&ts=' + now`; set `X-Myjd-Appkey` header; `xhr.timeout = 5000`; `xhr.send()`
   - Call `startMouseMoveReporting(callbackUrl)` in main flow (after sendLoadedEvent, inside isJdLocalhost gate)

6. **Update main flow** — After the existing `startCountdown(callbackUrl)` call, add a block gated on `isJdLocalhost`:
   ```javascript
   // JD protocol callbacks (localhost only)
   if (isJdLocalhost) {
       canCloseHandle = startCanClosePolling(callbackUrl);
       sendLoadedEvent(callbackUrl);
       startMouseMoveReporting(callbackUrl);
   }
   ```

7. **Update `beforeunload` cleanup** to also clear `canCloseHandle`:
   ```javascript
   window.addEventListener('beforeunload', function() {
       if (pollingHandle) clearInterval(pollingHandle);
       if (countdownHandle) clearInterval(countdownHandle);
       if (canCloseHandle) clearInterval(canCloseHandle);
   });
   ```
