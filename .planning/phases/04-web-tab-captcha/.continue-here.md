<context>
Deep research revealed the full JDownloader CAPTCHA flow:
1. User sends download link to JDownloader
2. JDownloader's HOSTER PLUGIN processes the page, encounters CAPTCHA
3. Plugin extracts siteKey, creates RecaptchaV2Challenge
4. BrowserSolver opens http://127.0.0.1:24613/captcha/recaptchav2/HOSTER/?id=X in browser
5. JDownloader's own HTML page renders the CAPTCHA widget with correct siteKey
6. Content script enhances page with skip buttons + countdown
7. User solves → token submitted back to JDownloader via callback URL
8. JDownloader's JS AND content script both submit token (first wins, harmless duplicate)
